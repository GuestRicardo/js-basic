Nodemon e Sucrase

Nodemon

O nodemon é um utilitário que monitora as mudanças nos arquivos do seu projeto e reinicia automaticamente o servidor Node. js quando necessário. Assim, você não precisa parar e iniciar o servidor manualmente a cada modificação.

Sucrase

Fala Galera beleza?!Como deve saber para se trabalhar com Node precisamos "puxar" módulos e usá-los na nossa aplicação e para isso usamos a função require( ). Mas já pensou se pudesse importar esses módulos do Node e bibliotecas de outros usando o import/export do ECMA?
A biblioteca sucrase que faz exatamente isso,
ela permite que voçê use o import e export no projeto em desenvolvimento e faz todas as mudanças necessárias quando o seu projeto for para a produção e o melhor de tudo que sua configuração é bem simples.

Instalando os 2 de uma vez só:

npm install nodemon sucrase --save-dev

Pós a instalação o nodemon precisará reconhecer o sucrase, abaixo esta sendo feito:

1 - Na raiz do do projeto com botao direito do mause será preciso criar um arquivo com nome nodemon.json
2 - dentro do arquivo ira colocar a seguinte configuração:
3 - depois será preciso entrar no package.json e adicionar dentro de scripts
"dev": "nodemon server.js"
4 - o proximo passo é adicionar o arquivo server.js dentro do projeto."para q o script acima possa executa-lo"
Para questa de testes, irei fgazer import e export no arquivo do server.js
Mas antes disso será preciso executar o server, para isso é so digitar no terminal:
npm run dev
depois de com nodemon em execução será seguido os passos abaixo para o teste.
