Mas o execução tem fim, se nao determinar se colocar para rodar ira rodar ate vc manda para

setTimeout

É a muito parecida com a setInterval a diferença é que ela executa somente uma vez.

Exemplo:

Aqui é o mesmo exemplo, so mudou q a função executará somente uma vez em 1 segundo
function mostradorHora(){
let data = new Date();
return data.toLocaleTimeString('pt-BR',{ hour12: false });
}
const timer = setTimeout(()=>{
console.log(mostradorHora());
}, 1000);
