Funções

Funções serve para executar ações, é um bloco de codigo para executar um codigo q envia um valor e depois quando precisar pega um valor de volta, tambem existem outras q nao retornam valores.

function nomedaFuncao (recebe alguma regra, parametros ou atributo ) {
corpo( escopo ou corpo )
}
nomedaFuncao( para enviar parametros ) aq esta sendo chamada(executada)

Exemplo:
function saudacao(nome){
console.log(`Bom dia ${nome}!`);
}
saudacao('Ricardo')

nesse exemplo nao esta sendo retornado nada, para q ela tenha algum retorno ela precisa ser armazenada em alguma variavel.
Exemplo:

function saudacao(nome){
return `Bom dia ${nome}!`;
}
const qualquervariavel = saudacao('Ricardo')
console.log(qualquervariavel)

A variavel esta recebendo o q a função enviou


Arrow function

É uma forma abreviada de apresentar uma função, deixando ela em uma so linha

exemplo:

const raiz = n => n ** 0.5;
console.log (raiz(9));
