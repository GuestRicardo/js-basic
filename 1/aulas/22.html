Heranças com Classes

Herança é um princípio de orientação a objetos, que permite que classes compartilhem atributos e métodos, através de "heranças". Ela é usada na intenção de reaproveitar código ou comportamento generalizado ou especializar operações ou atributos.

Super class
class DispositivoEletronico{
constructor(nome){
this.nome = nome;
this.ligado = false;
}
//metodos
ligar(){
if(this.ligado){
console.log( this.nome + 'já ligado');
return;
}
//estado
this.ligado = true;
}
desligar(){
if(!this.ligado){
console.log( this.nome + 'já desligado');
return;
}
//estado
this.ligado = false;
}
}
//herança(So a linha abaixo ja faz herdar tudo da classe principal)
class Tv extends DispositivoEletronico{
constructor(nome, cor, modelo){
//AQ esta sendo pego o nome q esta na superclass
super(nome);
this.cor = cor;
this.modelo= modelo;
}
}
const dispositivo1 = new Tv ('TV samptoshiba', 'preto', 'A5226');

dispositivo1.ligar();
