Promises

O Promise (objeto construtor) representa a eventual conclusão (ou falha) de uma operação assíncrona e seu valor resultante.

Descrição

A Promise é um proxy para um valor não necessariamente conhecido quando a promessa é criada. Ele permite que você associe os manipuladores ao valor de sucesso ou ao motivo da falha de uma ação assíncrona. Isso permite que métodos assíncronos retornem valores como métodos síncronos: em vez de retornar imediatamente o valor final, o método assíncrono retorna uma promessa de fornecer o valor em algum momento no futuro.

A promises pode esta em um destes estados:

  • pendente: pendente : estado inicial, nem cumprido nem rejeitado.
  • concluida: significa que a operação foi concluída com sucesso.
  • rejeitada: significa que a operação falhou.

Exemplo:

//função para gerar numeros aleatorios

function aleatorio(min, max){
//convertendo para segundos
min *= 1000;
max *= 1000;
return Math.floor(Math.radom() * (max - min) + min);
}

function esperaAi(mensagem, tempo){
setTimeOut(()=>{
console.log(mensagem);
}, tempo);
}
esperaAi('frase1', aleatorio(1, 3));
esperaAi('frase2', aleatorio(1, 3));
esperaAi('frase3', aleatorio(1, 3));
