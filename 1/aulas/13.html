Varias formas de se declarar uma função

Declaração literal

teste();pode ser chamada aq

function teste(){
return executa a ação;
}

teste();ou aq
Neste tipo de função será vista como function hoisting isso significa que ela estará sempre ao topo, podendo ser chamada no inicio ou no fim do codigo

Funções como expressões

É quando uma função vem como uma variavel, seria criar uma função como um dado
const exemplo = function (){
return ação;
}
exemplo();
outro exemplo; uma função executando outra função como paramametro.
function executaFuncao(funcao){
funcao(); //neste exemplo foi executado a função q esta como variavel acima
};
executaFuncao(exemplo); aq esta sendo pego a variavel e esta sendo adicionado na função que esta dentro da executaFuncao

Usando em Objetos

aqui ela esta sendo usada como um metodo do objeto
const objeto = {
falar: function (){
console.log('estou aq');
}
};
objeto.falar();
tambem pode ser feito dessa forma q tambem se considera certo e funciona da mesma forma.
const objeto = {
falar(){
console.log('estou aq');
}
};
objeto.falar();
