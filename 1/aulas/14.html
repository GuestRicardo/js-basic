Getters e Setters

getters e setters q seria fazer o metodo do objeto simular q é uma propriedade, eles sao uma maneira de protejer.

o gett para obter (pegar) ele serve para pegar o valor e exibir, o setter para setar(colocar) para configurar o valor, ele modifica o valor e valida
exemplo de get e set

Fuctions
function criaProduto(nome){
return {
get nome(){
return nome;
}
set nome(valor){
nome = valor;
}
};
};
const p1 = criaProduto('cardaço de sapato');
console.log(p1.nome);
Class
class Pessoa {
constructor(nome, sobrenome){
this.nome= nome;
this.sobrenome= sobrenome;
}
//essa função funciona como um atributo nao uma função
get nomeCompleto (){
return this.nome + '' + this.sobrenome;
}
set nomeCompleto(valor){
valor = valor.split('');
this.nome = valor.shift();
this.sobrenome = valor.join('');
}
}
const p1 = new Pessoa('Maria', 'Jose');
p1.nomeCompleto = 'Maria Jose Pinto'; //seter
console.log(p1.nomeCompleto); //getter
