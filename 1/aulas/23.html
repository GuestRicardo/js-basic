Métodos

Definição de métodos em inicializadores de objetos. É uma abreviação para uma função atribuída ao nome do método.

Documentação

Métodos de instancia

OBSERVAÇÂO: referente a instancia

São os mais tradicionais, eles são instanciados pelo .
exemplo: controle.aumentarVolume();
tambem se usa a palavra new para chamar o construtor da classe, e quando se instancia a classe esta sendo criado um novo objeto.

Métodos estaticos

OBSERVAÇÂO: referente a classe

São metodos que vc consgui acessar na classe sem instanciar a classe, NÂO ultilizar a palavra new, simplismente são como funções normais detro das classe, eles estao referentes somente a classe, q ele se encontra e não a instancia da classe.
Em classe usa se a palavra static para se referenciar q o metodo é estatico dentro da classe. eoutra coisa, este metodo static nao tem acesso aos dados da instancia

class ControleRemoto{
constructor(tv){
this.tv = tv;
this.volume = 0;
}
//metodos de instancia
aumentarVolume(){
this.volume += 2;
}
diminuirVolume(){
this.volume -= 2;
}
//metodos de estatico
static trocarPilha(){
console.log('Ok, pilha trocada')
}
}
const controle = new ControleRemoto('LG');
//****todos estao sendo instanciados***// controle.aumentarVolume();
controle.diminuirVolume();
//****nao precisa da palavra new***//
ControleRemoto.trocaPilha();
