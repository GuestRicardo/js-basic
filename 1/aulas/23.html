Metodos

Definição de métodos em inicializadores de objetos. É uma abreviação para uma função atribuída ao nome do método.

Métodos de instancia

Métodos estaticos

São metodos que vc consgui acessar na classe sem instanciar a classe, ultilizar a palavra new, simplismente são como funções normais detro das classe

class ControleRemoto{
constructor(tv){
this.tv = tv;
this.volume = 0;
}
//metodos
aumentarVolume(){
this.volume += 2;
}
diminuirVolume(){
this.volume -= 2;
}
}
