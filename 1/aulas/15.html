Prototype

JavaScript é baseado em prototipos(prototype) para passar propriedades e metodos de um objeto para outro.

Definição de Prototype(prototipo)
protodtipo é um trermo usado para se referir ao que foi criado pela primeira vez, servido de modelo ou molde para futuras produções.

Todos os objetos tem uma referencia interna para um prototipo (_proto_) que vem da propriedade, prototype da função construtoraque foi usada para cria-la. quando tenta acessar um membro de um objeto, primeiro o motor do js vai tentar encontrar este membro no proprio objeto e depois a cadeia de prototipos é usada ate o topo(NULL) ate encontrar (ou nao) tal membro.

Esta forma é sem o uso do prototype, sendo que dessa forma é menos performatico (sendo um desperdicio de recurso do pc do cliente), para isso existe o prototype(via prototipos)
