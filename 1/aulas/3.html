Objetos

Um objeto é um elemento computacional que representa, no domínio da solução, alguma entidade (abstrata ou concreta) do domínio de interesse do problema sob análise. Objetos similares são agrupados em classes. No paradigma de orientação a objetos, tudo pode ser potencialmente representado como um objeto.

Objeto Literal

segue abaixo 2 objetos simples

const pessoa = {
nome:'Maria',
sobrenomenome = 'Joaquina',
idade = 45
};

console.log(pessoa.nome)

Arrays

Arrays é uma Lista ou uma coleção de dados, é melhor forma de manipular dados, sendo q pode se usar os idices pra facilitar.
exemplo:
const alunos = ['joao','maria', 'eduarda', 'juana', edilson]
1- E para acessar um indice.
exemplo:
console.log =(alunos[4])

2- Se caso queira editar um indice, substituir.
exemplo:
alunos[4]= Leoncio

3- ADICIONANDO indice. neste caso é so add em um indice inexistente
exemplo:
alunos[5]= Juarez

outra forma, a forma ideal.
alunos[alunos.length] = 'maria isabel'
alunos[alunos.length] = 'joana'
alunos[alunos.length] = 'Roberta'

A forma adequada, a q todos usam é usando push sendo q esse exemplo adiciona o elemento no fim
exemplo:
alunos.push('Otavio');

Para adicionar um item no inicio é so usar unshift
exemplo:
alunos.unshift('Clesio');

4- Para saber o tamanho do array
exemplo:
console.log(alunos.length)

5- Para remover um indice do array
exemplo de remoção pelo fim, aq automaticamente o ultimo elemento do indice sera removido:
alunos.pop()

alem de remover, da para salvar o elemento removido em uma varialvel
const removida = alunos.pop()

exemplo de remoção pelo inicio, aq automaticamente o primeiro elemento do indice sera removido:
alunos.shift()

se caso queira remover o indice e deixalo vazio,
delete.alunos[3]
neste exemplo, juana ira desaparecer, e nao sera substituida pelo indice seguinte

6- Pegando varios elementos de uma vez, fatiando o elemento, é similar ao exemplo das strings
exemplo:
console.log(alunos.slice(1,4))
sera exibido entre virgula os elementos dos indices.

7- Para descobrir se o elemento usado é um array.
exemplo:
console.log(alunos instanceof Array);
neste caso ira retornar:
object
true pq alunos é um array

8- Para criar uma copia de um array de forma pratica.
exemplo:
let a =[1, 2, 3]
let b =[...a]


Objeto Date

Date é uma função construtora, e para usar o objeto date é sempre obrigado a usar a palavra new pq o Date é uma função construtora, e sempre ela começara um a primeira letra maiuscula a diferenciando
exemplo:
const data = new Date()
Neste esxemplo esta sendo criado um novo objeto e esta sendo locado na variavel data, q por sua vez quandp for chamada será exibida todos os dados de hras no momento q for usa-la.

Agora será mostrado os principais usabilidade da Date

Exibindo em formato gringo:

const data = new Date();
console.log(data.toString());

Para pegar a data no momento

console.log(Date.now); aqui pegará todos os dados de uma vez só, exibira todas as informações
(tambem sera exibido a todos os milesimos de segundos percorridos ate hje desde o dia zero)

E se quiser pegar a primeira data exibida pelo Objeto:
exemplo:
const data = new Date(0); //aq sera exibido 01/01/1970 q é a epoca unix o marco 0, se queira data inferior é so usar valor negativo e se quiser positivo é so usar numeros positivos
E para usar as datas é preciso usar em milisegundos e o calculo para usar em hroas é:

Hora

const 1 hora = 60 segundos * 60segundos * 1 * 1000 milisegundos

Dia

const 1 dia = 60 segundos * 60segundos * 24 * 1000 milisegundos

Para exibir data especifica

const data = new Date(ano, mes, hora, minutos, segundos, milisegundos);

const data = new Date(2019, 3, 20, 15, 19);
console.log(data.toString());
será exibido: 2019, 3 abril, 20:15,19

usando Strings:

const data = new Date('2019-04-20 20:20:32');
será exibido: 2019, 20 abril, 20:20,32

Formatando uma data usando uma função

const data = new Date();
function formatandoData(data){

const dia = new Date("Dia", data.getDate());

return `${dia} / ${mes} / ${ano} ${hora} : ${minutos} , ${segundos}` }
