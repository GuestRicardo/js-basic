Fetch Api (GET)

A Fetch API fornece uma interface JavaScript para acessar e manipular partes do protocolo , como solicitações e respostas. Ele também fornece um método global fetch()que fornece uma maneira fácil e lógica de buscar recursos de forma assíncrona na rede.
Ao contrário XMLHttpRequestde uma API baseada em retorno de chamada, o Fetch é baseado em promessa e fornece uma alternativa melhor que pode ser facilmente usada em service workers . O Fetch também integra conceitos avançados de HTTP, como CORS e outras extensões para HTTP.
