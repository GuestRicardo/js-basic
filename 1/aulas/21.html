Classes

Em js classe é a mesma coisa da função construtora, a diferença é o modo de escrever, elas fazem a mesmoa coisa.
Na classe ela carrega um metodo especial chamado Constructor

Classe
class Pessoa {
constructor(nome, sobrenome){
this.nome = nome;
this.sobrenome = sobrenome;
}
//Metodos
falar(){
console.log(`${this.nome} esta falando.`);
}
comer(){
console.log(`${this.nome} esta comendo.`);
}
beber(){
console.log(`${this.nome} esta bebendo.`);
}
};
const p1 = new Pessoa('Maria', 'Campos');
console.log(p1);

Função Construtora
function Pessoa (nome, sobrenome ){
this.nome = nome,
this.sobrenome = sobrenome;
};
//Metodos, já usando o prototype
Pessoa.prototype.falar = function(){
console.log(`${this.nome} esta falando.`);
};
Pessoa.prototype.comer = function(){
console.log(`${this.nome} esta comendo.`);
};
Pessoa.prototype.beber = function(){
console.log(`${this.nome} esta bebendo.`);
};
const p1 = new Pessoa('Jose','Maria');
console.log(p1)
