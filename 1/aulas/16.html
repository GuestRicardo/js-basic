Copiando Objetos

Forma Tradicional
const produtos = {nome: 'camisinha', preco: 6.0};
const copia = produtos;
Spread
copia = {...produto};
object.assign(target:(que seria um objeto vazio), Objetoque sera copiado) É um método estático que copia todas as propriedades próprias enumeráveis ​​ de um ou mais objetos de origem para um objeto de destino . Ele retorna o objeto de destino modificado.

Exemplo:

const caneca = Object.assign({}, produtos);
Ele copiara tudo do objeto(produto) para o target, q no caso sera vazio
const caneca = Object.assign({}, produtos, {material: 'porcelana'});
aq esta um exemplo se caso queira copiar e add outros objetos
