Operação Ternário

Serve paara substituir o if e else em situações mais simples e menos complexa
sintaxe:
const variavel = (condição >= o q deseja comparar) ? 'se for verdadeira rodara esse resultado' : 'se caso for falso';

exemplo:
const pontuacao = 999;
const nivelUsuario = pontuacao >= 1000 ? 'Usuario VIP' : 'Usuario Normal';

tambem existe uma forma mais simplificada aidna. setando um valor padrao para uma variavel.
exemplo:
const variavel1 = null;
variavel = variavel1 || 'fallback';
neste exemplo: será pego a variavel1 q é null e vai comparar com o resultado entre aspas, entao a variavel podera receber somente um dos dois resultados. resultados
