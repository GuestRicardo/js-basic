Revisão JS

Este arquivo foi feito, para armazenamento de minha revisão em js

1* Variaveis o js aceita varios tipos de variaveis, segue abaixo os 2 mais usados

let e var = let é a forma mais popular, var ja é uma forma mais antiga, ja esta fora de uso
exemplo:

let nome = Jose

Toda variavel pode ser alterada durante o codigo, exemplo:

declarando

let nome; //neste caso a variavelo esta sendo declarada sem nenhum valor, somente como nome

inicializando

nome = 'Maria Jose dos Santos' //a variavel de cima esta recebendo um valor

declarando e inicializando

let nome = 'Maria Jose dos Santos' //aq esta declarando e inicializando

Observação:
nunca redeclarar uma varial "LET" com mesmo nome de outra.
