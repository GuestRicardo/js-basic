5 - pesquisando por um indice é usado o indexOf() , exemplo a sera exibido o indice q a palavra começa
exemplo:
console.log(umaString.indexOf('texto')) =
sera exibido o indice q a palavra começa, q no caso seria o 3
Observação se caso a palavra q esteja buscando nao tenha na string sera exibido -1, como sinal de inexistencia
E se caso queira fazer uma busca pós o indice q queira:
exemplo:

console.log(umaString.indexOf('o', 3'))
= aq no caso, ira começar a verificação da string apos o indice 3, e sera exibido o valor 7 pois a letra esta na setima posição.
Tabem tem como fazer essa busca de forma invertida, "de tras para frente: com o lastIndexOf()
exemplo:
console.log(umaString lastIndexOf('o', 3)) =
neste exemplo ira exibir -1, pois nao existe o apartir da posição 3, pois ele vai começar a busca começando pelo 3 ate a posição 0
