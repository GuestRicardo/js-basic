Em js tudo pode ser verificado "Avaliado".

em js o valor false literalmente é false, sendo avaliado de cara como false é o unico literal, tambem existe outros valores q sao retornado como false, so q tem as condições

Esses sao considerados os valores FALSYS.
0, string vazia ' ', null, undefined, NaN
Qualquer coisa fora dessa regra é avaliada como True

Exemplo:
console.log('jose' && 'Maria'); true pois os 2 sao strings cheias, sendo q sera retornadpo maria por ser o ultimo resultado
console.log('jose' && ' ' && 'Maria'); false pois o and, sempre quando encontra um valor false ele encerra a busca, sendo retornado o ' '
