Math

É um objeto relacionado a matematica, ele aceita vairas funções aq especificarei somente as mais populares

let num1 = 9.5465416
Arredondado para baixo
let arredondando = Math.floor(num1);
Arredondado para cima
let arredondando = Math.ceil(num1);
Arredondado automaticamente para o numero mais proximo
let arredondando = Math.round(num1);

Para pegar o maior numero em varios numeros:
console.log(Math.max(1,5,466,525,24,88,4)); aq sera exibido 525
Para pegar o menor numero em varios numeros:
console.log(Math.min(1,5,466,525,24,88,4)); aq sera exibido 1
Para numeros aleatorios:
console.log(Math.random()); aq sera gerado um numero aleatorio entre 0 e 1
Gerando numeros aleatorios entre 5 e 10
const aleatorio = Math.round(Math.random() * (10 - 5) + 5); => Aq esta sendo gerado numeros aleatorios entre 5 e 10 e esta sendo arredondado para o valor mais proximo
Para elevar o numero a potência:
console.log(Math.pow(2, 10)) => 2 elevado a 10 q é igual a 1024
ou
console.log(2**10);

Para pegar a raiz quadrada:
console.log(Math.sqrt(25));
