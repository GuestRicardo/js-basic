WebPack(Babel)

Babel é um pacote que tem disponivel via webpack, ele serve para converter codigos, de recentes para antigos, para navegadores antigos possam reconhecer o js

Instalando:
  • primeiro passo é instalar o webpack
  • segundo passo abra o terminal, locallize a pasta onde esta sendo usado
  • terceiro passo digite:
    npm install --seve-dev @babel/cli @babel/preset-env @babel/core
  • quarto passo será para criar o arquivo q será salvo as informações convertidas, Observação: lembrar localizar o local da pasta onde queira criar o arquico bundle.js que no caso é na pasta do arquivo q esta o js
    npx babel nomedoarquivo.js -o bundle.js --presets=@babel/env
    apos a execução do codigo acima sera craido o arquivo bundle.js com a versao antiga do codigo

    • Observação: apos a criação do bundle.js será preciso carrega-lo no lugar q queira usa-lo via script, apartir dai, todos navegadores reconheceram o codigo

    FORMA MANUAL: Toda mudança que for preciso tera que ser feito no arquivo js, sem ser o bundle, apos todas as mudanças, teram q refazer o mesmno processo
