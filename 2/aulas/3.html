Modelo

1 passo:

npm install --save-dev @babel/preset-env @babel/core @babel/cli babel-loader webpack webpack-cli regenerator-runtime core-js@2

Após a instalação sera criado o package.json e esta será o iniciador de tudo, se caso precise deinstalar as mesmas configurações em outra maquina é so copiar o arquivo package.json e colar na pasta onde for instalar o trabalho e apos colar o arquivo so sera preciso digitar no terminal npm install

2 passo:

Dentro da pasta onde foi instalado o projeto crie um arquivo com nome de webpack.config.js

Este arquivo sera responsavel por todas as configurações do webpack,
