Session

session é para salvar coisas do navegador do cliente, é quando salva infomações do navegador no servidor, tipo uma conexao de uma rede social, e é salvo em cookies. e se ativar o modo restrito, nao será usado as sessions, geralmente todo mundo usa os cookies

Flash Messages

sao mensagens q sao salvas na sessao, quando e preciso ter um poup app, exemplo: quando erra uma senha e aparece q esta errado, ai depois some

Instalando

npm install express-session connect-mongo connect-flash
No arquivo server.js
Adicionando session no projeto
const session = require('express-session');
Para salvar as sessions
const MongoStore = require('connect-mongo')(session);
Mensagem
const flash = require('connect-flash');
