Mas se fomos criando esta forma de rota ira ocupar muito espaço no codigo, e para resulmir as rotas do site existe uma propriedade do proprio express para resulmir as rotas de forma organizada.

Routers

Arquivo routes.js
Primeiro passo é importar o express e o router para o arquivo
const express = require('express');carrega o express
const route = express.Router();var para utilizar o router
O Router será responsavel pelas rotas, ele será encarregado de ver a rota e chamar o Controller para decidir qual viwer ou como modo irá ultilizar

const express = require('express');

Agora o proximo passo é exportar o route para o app.js para que possa exibir, para isso será preciso sobressair o export, q ja foi usado para exportar para o controllerHome.js, exemplo:
const express = require('express');
const route = express.Router();
const homeController = require('./controllers/homeController');

route.get('/', homeController.paginaInicial);

module.exports = route;
Agora todas as rotas q forem criadas passaram por esse arquivo, e os controllers na pasta de conttrollers
Agora no server.js terá q importar o route q foi exportado anteriormente
exemplo:
const express = require('express');
const app = express();
const routes = require('./routes')

app.use(routes);
