Models

O Model realiza a operação matemática e retorna o valor calculado para o Controller, que também é o único que possui conhecimento da existência da camada de visualização. Tendo o valor em mãos, o intermediador o repassa para a interface gráfica que exibirá para o usuário. Caso esta operação deva ser registrada em uma base de dados, o Model se encarrega também desta tarefa.

Model ou Modelo  Essa classe também é conhecida como Business Object Model (objeto modelo de negócio)..   Sua responsabilidade é gerenciar e controlar a forma como os dados se comportam por meio das funções, lógica e regras de negócios estabelecidas Ele é o detentor dos dados que recebe as informações do Controller, válida se ela está correta ou não e envia a resposta mais adequada. 

Exemplo de model

No arquivo server.js

Apos a adição do dotenv será preciso importar o mongoose,
const require('mongoose')
Agora será preciso conectar ao db
mongoose.connect(process.env.CONNECTIONSTRING)

//para saber q o sinal seja emitido apos a conexao do db .then(() => {
app.emit('pronto');
}).catch(erro=> console.log('ERRO na conexao', erro));
A variavel CONNECTIONSTRING foi criada em um arquivo separado, o .env, q esta sendo uma forma de resguardar os dados q estao sendo enviado para o git e as informações de chaves

No arquivo .env

Neste arquivo esta sendo armazenado as chaves de segurança para conexao no DB

CONNECTIONSTRING=
mongodb+srv://gestricardo:ricardo22@cl...
E o ele foi importado no server.js
require('dotenv').config();

No arquivo homeController.js

Agora no controlador, será preciso importar o homeModel
const HomeModel = require('../models/homeModel');
Agora será criando elemento q sera add no DB, ele retornará uma promise, e retornará com elemento completo, com ids e tudo
HomeModel.create({
titulo: 'Titulo para o primeiro teste',
descricao: 'Descrição para o primeiro teste'
})
Agora será capturado os elemnetos criados acima
.then(dados => console.log(dados))
.catch(erro => console.log(erro));

No arquivo homeModel.js

Agora será ciado o Model, sendo q neste processo este seria o segundo a ser criado, para q hovesse as importações

Primeiramente precisaremos do esquema
const HomeSchema = new mongoose.Schema({
titulo: {
type: String,
require: true
},
descricao: String
});
Mais antes de tudo para q funcione será preciso a importação do Mongoose
const mongoose = require('mongoose');
Agora sim o Model
const HomeModel = mongoose.model('Home', HomeSchema);
module.exports = HomeModel;
