readdir

lista os arquivos e subdiretórios em um diretório.

readdirSync

é usado para ler de forma síncrona o conteúdo de um determinado diretório. O método retorna uma array com todos os nomes de arquivo ou objetos no diretório. O argumento options pode ser usado para alterar o formato no qual os arquivos são retornados do método.

Parâmetros: este método aceita dois parâmetros mencionados acima e descritos abaixo:
  • path: contém o caminho do diretório de onde o conteúdo deve ser lido. Pode ser uma String, Buffer ou URL.
  • options: É um objeto que pode ser usado para especificar parâmetros opcionais que afetarão o método. Possui dois parâmetros opcionais:

    • encoding: É um valor de string que especifica qual codificação seria usada para os nomes de arquivo dados ao argumento de retorno de chamada. O valor padrão é 'utf8'.

    withFileTypes: é um valor booleano que especifica se os arquivos seriam retornados como objetos fs.Dirent. O valor padrão é falso'.

    Retorna: Retorna um array de objetos String, Buffer ou fs.Dirent que contém os arquivos do diretório.
