Imports

Tudo criado aq esta no arquivo app.js

Aqui sera onde estará todos os exports do mod1.js

const mod1 = require('./mod1');
aqui esta sendo importado tudo que tem no mod1, variaveis, funções, objetos, etc..
console.log(mod1.fala());
aq esta sendo executada no back do navegador a função q tem no mod1.js
const fala = mod1.fala;
este caso é quando quiser usar alguma função ou elemento q estiver no mod1, pois neste exemplo foi importado o modulo inteiro
const fala = require('./mod1').fala;
aqui esta sendo importado so a função fala
console.log(fala());
nao precisara mais da palavra mod1
