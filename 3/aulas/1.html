Imports

Tudo criado aq esta no arquivo app.js

Aqui sera onde estará todos os exports do mod1.js

const mod1 = require('./mod1');
aqui esta sendo importado tudo que tem no mod1, variaveis, funções, objetos, etc..
console.log(mod1.fala());
aq esta sendo executada no back do navegador a função q tem no mod1.js
const fala = mod1.fala;
este caso é quando quiser usar alguma função ou elemento q estiver no mod1, pois neste exemplo foi importado o modulo inteiro
const fala = require('./mod1').fala;
aqui esta sendo importado so a função fala
console.log(fala());
nao precisara mais da palavra mod1
const {nome, sobrenome, fala} = require('./mod1.js');
Aqui esta sendo via destruturação
const { Pessoa } = require('./mod1');
Aqui esta sendo importado Class

Usando modulos de formas separadas

Aqui veremos a forma de usar o modulo em forma separada, em condições que esteja tudo separado com funções definidas para cada um,

Este module vai so exportar uma função

EXPORTANDO
module.export = function (x,y){
return x * y
}
IMPORTANDO
const multiplicacao = require ('./mod1');
console.log(multiplicacao(2, 2));
*esta situação é bem parecida com a default usado no ES6, sendo que esta sendo usado somente uma função

Este exemplo acima vale parea qualquer situação, podendo ser somente uma variavel, ou sendo uma soma, etc...
