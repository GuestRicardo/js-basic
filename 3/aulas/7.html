Server Express

Passo a Passo

1- npm init -y

2- install express

3- Primeira coisa depois de instalar qualquer framework é preciso importar o modulo para o projeto.

Dentro do arquivo server.js
const express = require('express');

4- Agora é carrega-lo, por convenção é criado uma variavel chamada app

const app = express();

5-Agora é utiliza-lo, neste exemplo iremos ultiliza-lo para carregar rotas usando o GET(é responsavel pela entrega da página)
Ele seria o / no endereço web
Exemplo:

http://meusite.com/sobre
neste exemplo o GET esta entregando a pagina contato
E ultilizaremos a operação CRUD para criamos as rotas
app.get('/ROTA',(requisiçãoo q esta sendo pedido, respostaé o q vou mandar para o servidor){
resposta.send('testando o server');
})

6- Agora é preciso fazer com q o servidor ouça o que esta sendo enviado na porta do meu computador, e para isso é preciso passar a porta q deseja q ele vigie

app.listen(3000);

7- agora para executar o servidor, é so ir no terminal e digitar:

npm server.js
