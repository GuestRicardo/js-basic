Views

Ela serve para renderizar o html na tela usando o express

Arquivo server.js

Primeiramente é preciso setar o View no server.js para poder usa-lo

1 primeiro é preciso add o path
const path = require('path');
2 segundo aq estamos definindo o local absoluto onde esta os views
app.set('views', patch(__dirname, 'src', 'views'));
Agora é preciso definir a egine que vai renderizar o html
neste caso iremos usar o ejs, aq estamos usando ja a egine.
app.set('views egine','ejs');
Agora é preciso instala-la em nosso projeto, so ir no terminal e escrever:
npm install ejs

Arquivo homeController.js

Para renderizar o index.ejs q esta dentro da pasta views, é preciso exportar o index, exemplo:

div.exports.paginaInicial = (req, res)=> {
res.render('index')
}
