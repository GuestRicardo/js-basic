Paramêtros Express

Parametro em uma URL

caso queira add o parametro...
/:parametro1?/:parametro2:/

Acessendo parametros

Para acessar os parametros ira usar o req
req.params

Para o envio de requisições, da para pegar as informações enviadas usando o params(q sao os parametros) exemplo:

usando req.params

este req.params é a parte que vem na rota da URL, por exemplo, /, perfis, numeros etc...

app.get('/nomeDaRota:aq sera a chave q o params ira receber',(req, res)=>{
console.log(req.params);
res.send('testando arquivo enviado')
})

Para q o parametro seja opcional é so usar o '?' apos a chave q o params vai receber

app.get('/testes/:idUsuarios?', (req, res)=>{
console.log(req.params);
res.send('teste');
})

Quando se ve ou se passa as informações via URL é chamado de query String

req.query
exemplo:
localhost:3000/testes/?nome=maria&sobrenome=jose

Enviando e pegando query strings

Envio
localhost:3000/testes/?nome=maria&sobrenome=jose
se caso queira q exiba no seu backend somente uma das informações(coisas especificas), exemplo:
Pegando
app.get('/testes/:idUsuarios',(req, res)=>{
res.send(req.query.nome);
})
no exemplo acima sera retornado um json, com o parametro nome

Como acessar o POST

O POST geralmente vem no Body da requisição(req) exemplo:

console.log(req.body);
Aqui esta sendo exibido no console da aplicação, sem nenhum tratamento adequado
Aqui esta sendo tratado a requisição do body
app.use(express.urlcoded({
extend: true;
}))
neste exemplo acima irá retornar em um objeto o q foi enviado via URL
apos o tratamento, aq retornará o q foi tratado
app.post('/', (req, res)=>{
res.send(`O que vc me enviou foi:${req.body.nome}`)
})
ira exbir diretamente na pagina o q enviar via o primeiro formulario
