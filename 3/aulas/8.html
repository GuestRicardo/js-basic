Acessendo parametros

Para acessar os parametros ira usar o req
req.params

Para o envio de requisições, da para pegar as informações enviadas usando o params(q sao os parametros) exemplo:

usando req.params

app.get('/nomeDaRota:aq sera a chave q o params ira receber',(req, res)=>{
console.log(req.params);
res.send('testando arquivo enviado')
})

Quando se ver as informações via URL é chamado de query String

Para q o parametro seja opcional é so usar o '?' apos a chave q o params vai receber

app.get('/testes/:idUsuarios?', (req, res)=>{
console.log(req.params);
res.send('teste');
})
