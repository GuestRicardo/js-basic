Escrevendo via FS

Para criar e escrever via FS é so usar o fs.writeFile

Exemplo:

Arquivo escrever.

const fs = require('fs').promises;
const path = require('path');
Aq esta sendo criado o arquivo
const caminhoArquivo = path.resolve(__dirname, '..', 'test.txt');
Aq sera escrito a "frase 1" no arquivo criado, o flag: w apagara tudo q estiver no arquivo, para q nao escreva nada por cima e o utf8 e o codigo pt br
fs.writeFile(caminhoArquivo, 'Frase 1', {flag: 'w', encoding: 'utf-8'});
o utf8 nao é obrigatorio mandar
